Array::Value

const Scalar::Temp operator[](int) const
Scalar::Lvalue operator[](int)

Get and/or set a value of an array element. If a value doesn't exist, undefined is returned.

void push(...)

Pushes one or more values onto the end of the array. The array will grow automatically to accommodate the addition. It accepts various kinds of primitives, as well as scalar and array values.

const Scalar::Temp pop()

Pops an SV off the end of the array. Returns undef if the array is empty.

const Scalar::Temp shift()

Shifts a scalar off the beginning of the array.

void unshift(..)

Unshift a scalar value at the beginning of an array.

const Scalar::Return_value remove(int)

Deletes the element indexed by key from the array. Returns the deleted element.

bool exists(int) const

Returns true if the element indexed by key exists.

unsigned length() const
Length length()

Returns the length of the array. It is similar to scalar @array in perl. It can be assigned to to change the length of the array.

void clear()

Clears an array, making it empty. Does not free the memory used by the array itself.

void undefine()

Undefines the array. Frees the memory used by the array itself.

void extend(unsigned size)

Pre-extend an array. size is the index to which the array should be extended.

template<typename T> Array::Temp map(const T& functor) const

This method accepts a function or function object as its argument. It applies this function to each of the values in the array and makes a new array from that. The functor must accept one argument, which can be of any type Scalar::Value can convert to. The functor's return value can be of any type that can be pushed onto an array. Also note that the argument is not passed as an lvalue. This is a design feature.

template<typename T> Array::Temp grep(const T& functor) const

This method accepts a function or function object as its argument. It applies this function to each of the values in the array and if the return value is true, it pushes the original value into the array it returns. The functor must accept one argument, which can be of any type Scalar::Value can convert to. The functor's return value can be of any type that implicitly converts to bool. Also note that the argument is not passed as an lvalue. This is a design feature.

template<typename T> void each(const T& functor) const
template<typename T> void each(const T& functor)

This method accepts a function or function object as its sole argument. It applies this function to each of the values in the array. The functor must accept one argument, which can be of any type Scalar::Value can convert to. Note that the function will be offered an alias of the value, so the non-const version can be used to modify the array's values.

template<typename T> void each_index(const T& functor) const
template<typename T> void each_index(const T& functor)

This method accepts a function or function object as its sole argument. It applies this function to each of the values in the array. The functor must accept two arguments; the first being an integer, and the second can be of any type Scalar::Value can convert to. Note that the function will be offered an alias of the value, so the non-const version can be used to modify the array's values.

Iterators

This class provides both forward and backward random access iterators. Iterators remain valid when values are changes or when values are added to the end, but will not remain valid when a value is added to the beginning or in the middle.
